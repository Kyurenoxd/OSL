## ✨ Features

- **Server Management**
  - Server voting system
  - Quick server switching
  - Favorites system

- **Statistics**
  - Playtime tracking for each server
  - Session tracking
  - History of last 5 played servers

- **Customization**
  - Dark/Light theme
  - Sorting preferences
  - Auto-close option

- **Multilingual Support**
  - English
  - Russian
  - Japanese
  - Portuguese

## 🛠️ Tech Stack

- Electron
- React
- TypeScript
- Tailwind CSS
- Vite
